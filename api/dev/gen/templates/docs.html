Authentication

All API endpoints require authentication via API key. Include your key as a query parameter:

?apikey=YOUR_API_KEY
Note: Keep your API key secure and never expose it in client-side code.

Error Codes

Code Message Description
400 Bad Request Invalid request parameters or malformed data
401 Unauthorized Missing or invalid API key
403 Forbidden Insufficient permissions
404 Not Found Requested resource doesn't exist
500 Internal Server Error Server encountered an unexpected condition

Available Endpoints

Endpoint Method Description Parameters Try It
/api/tables GET List all available tables in the database None
/api/<table> GET POST PUT DELETE Full CRUD operations for any table
  • action (required): list, view, insert, update, delete
  • editid1 (required for view, update, delete)
  • records (for list): Number of records (default: 100)
  • skip (for list): Skip records (default: 0)
  • q (for list): Advanced filter
  • qs (for list): Full-text search
/api/<table>/schema GET Get schema information for specified table None
